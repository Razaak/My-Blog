Add Class/Remove Class

Using .addClass(), and .removeClass() to trigger CSS3 Animations

In this example I will demonstrate one way in which an event, such as a click event, can be used to trigger animations by adding or removing a CSS class.

The HTML

We will start by creating an outside div with a class of .alpha, that will contain an image, another div with a class of .beta, and an a tag that will be used to trigger the animation.

<div class="alpha">

  <img src="http://placehold.it/400x250" />

  <div class="beta">

    <a href="#" id="trigger"></a>

  </div>

</div>

CSS Code

Here we add some styling and positioning to our HTML code. In .active and .open classes, we will add the animations. Both will trigger when the small white box (the a tag) is clicked and will be removed when clicked on again.


body {
    width: 960px;
    margin: 20px auto;
    background-color: #6e6e6e;
 }
 
.alpha {
    position: absolute;
    top: 50%;
    left: 50%;
    margin-left: -250px;
    margin-top: -150px;
}

.beta {
    background-color: #FFB2AF;
    display: block;
    width: 150px;
    height: 50px;
    position: absolute;
    bottom: 2%;
  }

a#trigger {
    float: right;
    display: block;
    width: 15px;
    height: 15px;
    margin: 12px;
    background-color: white;
}

/* CSS3 Animations */
.active {
    height: 150px;
    -webkit-transition: all cubic-bezier(0.86, 0, 0.07, 1) .5s;
}

.open {
    -webkit-transform: rotate(45deg);
    -webkit-transition: all cubic-bezier(0.86, 0, 0.07, 1) .5s;
}

JS Code

Inside the click event we add an if/else statement that first checks to see if the anchor tag has a class of open applied to it. Since we have not yet clicked on it, the if statement will return a false and move to the else block which will then apply the classes of .open and .active. If we click the anchor tag again, this time, the if block will trigger, removing the classes. The logic of this may seem unusual at first and there are of course other ways of doing it but this remains simple and logical. 


$('#trigger').on('click', function(event){ 
event.preventDefault();

  if ( $('#trigger').hasClass('open') ) {
    $('.beta').removeClass('active');
    $(this).removeClass('open');
  } else {
    $('.beta').addClass('active');
    $(this).addClass('open');
  }
});
