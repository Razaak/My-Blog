Math.Random

Using JQuery and JS to generate a random colour on Click

Here is a neat little puzzle that generates three random numbers between 1 and 256 and then uses these numbers to generate a random colour.

HTML Code

First we need some HTML that will be the target of these changes. A simple divider will do: 

<div class="box"></div>

CSS Code

Next we need to style the box. Here I gave it a width and height of 100px, positioned it in the center of the screen, and added a the border radius property to transform it into a circle.


 .box {
    position: absolute;
    top: 50%;
    left: 50%;
    width: 100px;
    height: 100px;
    background-color: #1f1f1f;
    border-radius: 50px;
    margin-left: -50px;
    margin-top: -50px;
}

JS Code

Here we create a click function that will trigger the code each time the divider is clicked. Next we create three variables each of which will store a random number from 1 to 256. The Math.floor is used to round out the number into a whole number so that no fractions will be picked. Finally a variable (randomRGBColor) containing the three nummbers concatinated with rpg() will be applied dynamically to the background-color of the div using the css selector.


$(".box").on("click", function() {
    var randomOne = Math.floor(Math.random() * 256);
    var randomTwo = Math.floor(Math.random() * 256);
    var randomThree = Math.floor(Math.random() * 256);

    var randomRGBColor = "rgb(" + randomOne + "," + randomTwo + "," + randomThree + ")";

  $(".box").css("background-color",randomRGBColor);
});
